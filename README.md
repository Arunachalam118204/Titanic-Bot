# 🚢 TitanicBot

An AI-powered chatbot to explore Titanic passenger data.

## Features
- Ask questions about passengers in natural language
- Generate charts and visualizations
- Powered by Groq AI + Streamlit

## How to Run
pip install streamlit pandas matplotlib seaborn pillow groq
streamlit run app.py

## Tech Stack
- Python, Streamlit, Groq API, Pandas, Matplotlib
